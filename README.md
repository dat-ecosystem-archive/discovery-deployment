# discovery
Deploy your own peer discovery infrastructure

- dns-discovery
- dat-gateway
- signalhubws
- signalhub
- any other peer discovery servers (dht bootstrap nodes?)

## Usage


ideas:
```
$ npm install -g dat-discovery

$ dat-discovery configFile.yml
```

or

```
$ git clone ..
$ ./run.sh
```

or supply a Docker container...
